Beli domain Id, satu hari langsung jadi

  • 21 September 2011

Update April 2017

Apakah domain .id itu ?

Sama dengan domain lain. Yang unik adalah: .id adalah singkatan untuk Indonesia.

Apakah keunggulan dari domain .id ?

Domain .id ada dua jenis, yaitu dengan satu titik seperti namasitus.id, ada yang dua titik seperti susetyo.web.id. Keunggulannya cukup banyak, di antaranya adalah:

Lebih ekonomis.

Pada umumnya harga domain berkisar 100 ribuan setahun. Domain .id juga sama. Namun jika anda mau menggunakan domain .id yang dua titik, harganya bisa separonya.

Keunggulan SEO.

Jika web anda berbahasa Indonesia, maka domain .id anda, akan menjadi nilai tambah. Mesin pencari seperti Google dan Yahoo akan mengutamakan domain .id, jika ada pencarian yang berhubungan dengan Indonesia.

Kebanggaan nasional.

Dengan semakin banyaknya warga Indonesia yang menggunakan domain .id, maka keberadaan bangsa Indonesia di internet akan semakin diperhitungkan dan berpengaruh.

Keamanan dan kepercayaan

Pada umumnya,domain lain bisa langsung digunakan setelah membayar. Tetai domain .id berbeda. Kita harus memberikan persyaratan identitas dan sejenisnya.

Tentunya pihak-pihak yang ingin menggunakan domain.id, untuk hal-hal buruk, akan berpikir berkali-kali.

Sejak awal kehadiran domain .id, masyarakat internet internasional sudah mengetahui hal itu.

Dari sudut pandang pengunjung web, akan menambah kepercayaan kepada pemilik situs.

Dalam jangka panjang, kita berharap, pengguna domain .id akan semakin dihormati.

Apa kerugian menggunakan domain .id ?

Kerugian ? Sejauh ini saya belum menemukan kerugian domain .id. Namun memang ada beberapa mitos, yang akan saya luruskan di bawah ini :

Mitos SEO

Ada yang bilang, domain .id sulit bersaing di mesin pencarian. Google enggan mengindeks domain .id, benarkah ?

Mesin pencari tidak dirancang untuk memperhitungkan SARA ( Suku, Agama, Ras atau Antar golongan). Faktor utama yang diperhitungkan adalah kualitas web dan kualitas isinya.

Kenyataan bahwa banyak pengguna yang sulit bersaing, disebabkan oleh beberapa hal:

Pengguna pemula

Murahnya domain .id, terutama yang dua titik, memancing minat para pemula untuk belajar. Dengan demikian, sangat mudah dijumpai di internet, keluhan para pemula ini. Bisa dimaklumi jika para pemula belum menguasai teknik-teknik dasar optimasi web.

Salah sasaran

Misalnya menggunakan domain .id, tapi isi web sama sekali tidak berhubungan dengan Indonesia. Atau malah tidak berbahasa Indonesia.

Ini mirip dengan domain .pw ( dari negara Palau ) atau .tk ( dari negara Tokelau ). Ada pengguna domain .pw dan .tk dari Indonesia, menggunakannya untuk sasaran wilayah Indonesia. Dan berbahasa Indonesia. Akibatnya ? Sulit bersaing dengan .id.

Plagiasi dan isi ganda.

Ada yang bukan pemula, pakar optimasi web, menggunakan domain .id. Tetapi juga mengeluh webnya sulit bersaing.

Ternyata setelah diselidiki lebih jauh, webnya penuh tulisan plagiat. Juga isi ganda ( duplicate content ) hasil copy paste dari web lain.

Tidak dapat dipungkiri, hal ini kebiasaan yang masih umum di Indonesia. Tentunya ini akan mempermudah Google untuk melemparkan web tersebut, ke peringkat yang paling bawah.

Mitos server

Domain .id servernya lambat ? Ini juga mitos ! Sebab domain “hanya” berurusan dengan DNS. Bisa dimuat langsung di memori, hampir tidak “menyentuh” harddisk sama sekali.

Umumnya memakan waktu kurang dari sepersepuluh detik. Selanjutnya tergantung server anda. Apakah server anda cepat atau lambat.

Harap dipahami bahwa DNS server dan web server adalah dua hal yang berbeda. Alias tidak sama. Jangan dicampuraduk.
